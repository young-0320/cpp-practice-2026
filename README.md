## 챕터별 연습 내용 (`practice/`)

### `CH04` 배열/문자열/구조체 기초

- `LAB_4.1`: 배열 입력/출력, 총점/평균 계산
- `LAB_4.2`: C 문자열에서 숫자 시작 전까지 잘라내기(`char[]`, `strncpy`)
- `LAB_4.3`: `std::string`으로 이름/학번 입력 처리
- `LAB_4.4`: `struct student` + `std::array`로 학생 정보 출력
- `LAB_4.5`: 실습 틀만 생성
- `LAB_4.6`: 동적 배열(`new[]`/`delete[]`)로 학생 정보 관리
- `LAB_4.7`: 포인터 연산으로 배열 원소 뒤집기
- `LAB_4.8`: `std::vector` 원소 접근(`at`), 크기/메모리 확인

### `CH05` 반복문/패턴 출력

- `LAB_5.1`: `for` 반복으로 1^2 ~ 100^2 합 계산
- `LAB_5.2`: 중첩 반복으로 구구단 출력
- `LAB_5.3`: 포인터 순회 + `toupper`로 대문자 변환
- `LAB_5.4`: 실습 틀만 생성
- `LAB_5.5`: 실습 틀만 생성
- `LAB_star`: 점(`.`)+별(`*`) 패턴 출력

### `CH06` 조건/문자 처리/파일 입출력

- `LAB_6.1`: 업다운 숫자 맞추기 게임(`while`, 분기)
- `LAB_6.2`: 문자열 정규화(첫 글자 대문자, 나머지 소문자/비문자 공백 치환)
- `LAB_6.3`: 강수 데이터 파일 생성/저장(`ofstream`)
- `LAB_6.4`: 파일 읽기 출력(`ifstream`, 줄 단위 읽기)

### `CH07` 함수와 포인터, 재귀

- `LAB_7.1`: 지수함수 급수 근사(테일러 합)와 오차 출력
- `LAB_7.2`: 배열 최대/최소/평균 함수 분리
- `LAB_7.3`: 2차원 배열 행 합 계산 후 동적 배열 반환
- `LAB_7.4`: 동적 문자열 생성 함수 구현(`new[]`)
- `LAB_7.5`: 직교좌표-극좌표 변환 함수(구조체 기반)
- `LAB_7.6`: 재귀로 1부터 n까지 합 계산

### `CH08` 함수 심화(인라인/기본인자/오버로딩/템플릿)

- `LAB_8.1`: `inline` 함수로 세제곱 합 계산
- `LAB_8.3`: 참조 전달 기반 좌표 변환 함수
- `LAB_8.4`: 기본 인자 함수(`sum(int n = 100)`) 사용
- `LAB_8.5`: `int`/`double` 절댓값 함수 오버로딩
- `LAB_8.6`: 함수 템플릿 + 사용자 정의 타입(복소수 구조체) 특수화

### `CH09` 분할 컴파일/링키지/네임스페이스/메모리 배치

- `LAB_9.1`: 헤더/구현 분리, 참조 반환 함수(`accumulate`)로 구조체 누적
- `LAB_9.2`: 외부/내부 링키지(`extern`, `static`)와 심볼 주소 비교
- `LAB_9.4`: placement new로 지정 버퍼에 객체 생성
- `LAB_9.5`: `pers`, `debts` 네임스페이스 분리와 사용 범위 제어

### `CH10` 클래스 설계 실습(`Student`)

- `LAB_10.1`: `Student` 클래스 선언, `enum class Status`, 성적/학기 데이터 모델링
- `LAB_10.2`: GPA 갱신 로직, 출력 포맷팅, 상위 GPA 학생 비교(`topGPA`)
- `student_test.cpp`: 다수 학생 데이터 생성/출력 및 최고 GPA 탐색 테스트

### `CH11` 연산자 오버로딩(`MyComplex`)

- `mycomplex.h/.cpp`: 복소수 클래스, `+/-/*`, `<<`, friend 연산자 구현
- `mycomplex_test.cpp`: 사칙 연산, 스칼라 곱, 크기(magnitude) 테스트

### `CH12` 동적 메모리/복사 제어/리팩터링

- `LAB_12.1`: `char*` 멤버를 가진 `Cow` 클래스(복사 생성자/대입 연산자/소멸자)
- `LAB_12.1_refactor`: `std::string` 기반으로 리팩터링해 Rule of Three 부담 완화
- `LAB_12.2`: 커스텀 `String` 클래스 구현(복사/대입/연결/입출력/대소문자 변환)
